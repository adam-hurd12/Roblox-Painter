# Roblox-Painter
Simple multiplayer game to showcase a custom grid-based item selection and placement system, developed using the Luau scripting language. Features basic UI and controls to demonstrate the capabilities of such a system. Based on the placement systems seen in games such as [Miner's Haven (Roblox)](https://www.roblox.com/games/258258996/Miners-Haven-Sandbox-Tycoon).
